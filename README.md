## **Deployment Instructions**

### Option 1: GitHub Pages
1. Create a repository on GitHub  
2. Upload the `wisdomania-game` folder  
3. Go to **Settings → Pages** → Set source to `main` branch / root folder  
4. Your site will be live at: `https://YOURUSERNAME.github.io/wisdomania-game/`  

### Option 2: Netlify
1. Sign up at [Netlify](https://www.netlify.com/)  
2. Drag & drop the `wisdomania-game` folder  
3. Site deploys automatically  
4. Optional: connect a custom domain (`wisdomania.app`, `.world`, `.com`)  
5. SSL is automatically enabled
